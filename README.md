# AWS Static Website Hosting Project

## PROJECT OVERVIEW
This project demonstrates how to host a static website on AWS using Amazon S3, CloudFront CDN, and HTTPS.

## SERVICES USED
- Amazon S3 (Static Website Hosting)
- AWS CloudFront (CDN)
- IAM
- 
##  ARCHITECTURE
IAM user --> S3 Bucket (Static Website) --> cloudfront distribution --> cloudfront URL

## IMPLEMENTATION STEPS
1. Created a unique S3 bucket
2. Enabled static website hosting
3. Configured bucket policy for public access
4. Created CloudFront distribution
5. Enabled HTTPS using ACM
6. Tested website using CloudFront URL
   
## KEY LEARNINGS
- Static website hosting on AWS
- CDN usage and performance improvement
- Securing website with HTTPS
- IAM and S3 permissions
